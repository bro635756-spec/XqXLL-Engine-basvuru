# 🚀 XQ Studio® Terminal & Recruitment System v1.0.2

**XQ Studio®**, gelecek nesil oyun teknolojileri ve **3.5D Spatial Engine** üzerine odaklanmış, bağımsız bir teknoloji oluşumudur. Bu depo, ekibe katılacak yetenekli geliştiriciler ve güvenlik uzmanları için geliştirilmiş **Terminal UX** tabanlı başvuru portalını içermektedir.

---

## 🛠 Teknik Altyapı & Standartlar

Proje kapsamında kullanılan teknolojiler ve geliştiricilerimizden beklenen yetkinlikler şunlardır:

*   **Programlama Dili:** sCs++ (s-Class C Plus Plus) - C performansında, modernize edilmiş bellek yönetimi.[cite: 1, 2]
*   **Grafik API:** OpenGL 4.6 ve Vulkan SDK entegrasyonu.
*   **Motor:** XqXLL 3.5D Engine (3D derinlik + Dinamik veri akışı).
*   **Hooking:** ScriptHookV.dll üzerinden RAGE Engine manipülasyonu.[cite: 1]
